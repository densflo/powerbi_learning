# True Reflection of Inventory (TRoI) — Documentation

This documentation covers the **True Reflection of Inventory (TRoI)** Power BI model, an enterprise IT infrastructure inventory system that consolidates data from 15+ enterprise source systems into a unified view of all servers and devices.

A secondary model (**VMware VMs Check**) is also documented.
